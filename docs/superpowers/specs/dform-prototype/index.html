强制覆盖
方向 A 关键决策： label 在控件上方（top-label），节省横向空间，长 label 不挤压控件； 章节用 01 / 02 / 03 mono 编号 + 细分隔线，无章节卡，视觉负担最低； 复杂类型展开后用 金色 2px 细 rail 标记嵌套范围，无填充背景，嵌套不"堆叠灰底"； 错误消息：字段下方一条小红字 + 红圆点前缀，无背景色； 加号按钮放在 label 行最右（margin-left: auto），不再绝对定位。
DIRECTION B

现代企业级

Label-left 网格（180px 右对齐）。章节作卡片，灰底章节头 + mono 序号 chip。 复杂类型展开为灰底 panel，每行嵌一张白底子卡。信息密度更高，结构感更强。
